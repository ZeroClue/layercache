# Review Archive

This directory contains all formal reviews for LayerCache specs and code.

## Archive Structure

```
docs/reviews/
├── README.md                  # This file
├── [date]-spec-[name].md      # Spec reviews
└── [date]-code-[phase].md     # Code reviews
```

## Naming Convention

- **Spec reviews**: `YYYY-MM-DD-spec-[spec-name].md`
- **Code reviews**: `YYYY-MM-DD-code-[phase-name].md`

## How to Add a Review

1. Create new file with appropriate naming convention
2. Use the review template from `../review-process.md`
3. Link the review from the spec document (if applicable)

## Review Agent

Reviews are generated by the Spec & Code Review Agent defined at:
- `.opencode/review-agent.md`

To invoke:
```bash
opencode task "Review the spec at docs/specs/v1.5.0-scale-context.md"
```
